1. Fun With Private Keys
	- why: desire to make own private keys, worry about retirement attacks & PRNGs
	- what to accomplish, not your keys

2. Being Paranoid
	- wealth journey, plan for 10-100x
	- attacks: phishing, compromised computer, exchange collapse, retirement, PRNGs

3. What are Private Keys
	- a very large random number, 2**256 == 10**77, visible universe contains 10**80 atoms
	- a wallet is a container for a private key, can derive everything else from that

4. How Are They Used
	- Private key -> Public Key -> Public Bitcoin Addesses (not reversible) using elliptic curve math
	- txns signed by private key, easily verified, not reversible

5. What Do They Look Like?
	- private keys can be represented in a variety of formats
	- Base2 (binary), Base10 (decimal), Base16(hexadecimal), Base58Check (letters[lower&capital] and numbers, minus (0, O, l, I))

6. BIPS - Bitcoin Improvement Proposals (quick note on how bitcoin evolves)
	- originally one private key/one address (non-deterministic), address starts with 1xx, P2PKH (Pay To PubKey Hash)
	- BIP016 - 3xx - P2SH (Pay To Script Hash)
	- bip084 - derivation path for P2WPKH (Pay To Witness Public Key Hash) SegWit Native, bc1xx
	- bip032/044 - hierarchical deterministic, extended public key (xpub), derivation path

7. Generating Entropy
	- draw words from hat, and calculate final seed word (SeedSigner or SeedPicker)
	- cards, coin flips, dice rolls, d16

	- show dictionary
	- ArmanTheParman, 11 bit number to dictionary
